The XDL_Module.cpp is used for moving all the CLB,DSP,RAMB instances from the specific XDL file.
If you want to generate the executable file, you should put this c++ file into the TORC environment and follow its tutorial to use 'make' to generate the executable file under LINUX.
Using command './XDL_Module <specific XDL file name> <X,Y operation like X+1,Y-1>' to move the instances.
